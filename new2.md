# PROPOSED METHODOLOGY (Flow Chart)

## System Workflow

Start  
↓  
User Registration & Login  
↓  
Profile Creation (Skills, Portfolio)  
↓  
Hackathon Discovery  
↓  
Team Formation / Join Request  
↓  

**Decision: Join Request Approved?**

- **If YES:**
  - Workspace Initialization  
  - Real-Time Communication (Chat Channels)  
  - Kanban Task Management  
  - Resource Sharing  
  - Project Submission  
  - End  

- **If NO:**
  - Review Feedback  
  - Update Profile  
  - Search Other Teams  
  - Retry Team Formation  

---

## Key Features in Workflow
- Centralized platform for collaboration  
- Real-time communication using Socket.IO  
- Task tracking using Kanban board  
- Iterative improvement through feedback loop
